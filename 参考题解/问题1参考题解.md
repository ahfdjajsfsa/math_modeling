# 一、问题1模型：基于活性密度的 E-DBSCAN 分区模型

## 1. 文献方法到本题的对应关系

| 文献中的对象 | 本题中的对象 |
|---|---|
| 频率 \(f\) | Manifold_X |
| 相速度 \(v\) | Manifold_Y |
| 能量值 \(E\) | Bioactivity_Score |
| 高频散能量区域 | 高活性热点区域 |
| 背景噪声点 | 普通或低活性分子 |
| E-DBSCAN 聚类 | 活性密度空间聚类 |

因此，我们将每个分子表示为：

\[
p_i=(x_i,y_i,A_i)
\]

其中：

\[
x_i=\text{Manifold\_X},\quad y_i=\text{Manifold\_Y},\quad A_i=\text{Bioactivity\_Score}
\]

---

# 二、分区依据

参考文献中的 E-DBSCAN，定义第 \(i\) 个分子的 \(\varepsilon\) 邻域为：

\[
N_{\varepsilon}(p_i)=\{p_j\mid d(p_i,p_j)\leq \varepsilon\}
\]

其中：

\[
d(p_i,p_j)=\sqrt{(x_i-x_j)^2+(y_i-y_j)^2}
\]

然后计算该邻域内的活性总和：

\[
S_i=\sum_{p_j\in N_{\varepsilon}(p_i)} A_j
\]

如果：

\[
S_i\geq MinActivity
\]

则认为该点是 **核心点**。

核心点以及由核心点连通起来的点组成一个 **高活性热点区域**。

其他点被划为普通区域。

---

# 三、模型流程

参考文献中的“能量阈值滤波 + E-DBSCAN 聚类”流程，本文问题1可以设计为：

1. **活性阈值滤波**  
   先去掉低活性背景点，只保留活性得分较高的分子参与热点识别。

2. **构造活性密度邻域**  
   对每个分子计算其 \(\varepsilon\) 邻域内的活性总和。

3. **判断核心点**  
   若邻域活性总和超过阈值，则该分子是核心点。

4. **区域生长聚类**  
   从核心点出发，把空间上相互邻近的高活性点连接成热点区域。

5. **区域比较**  
   比较热点区域和普通区域在活性得分、LogP、TPSA、MolWt 等理化指标上的差异。

# 四、简化版 Python 代码

下面这份代码比之前简单，核心就是自己实现一个 E-DBSCAN，不依赖复杂机器学习模型。

文件名假设为：

代码如下：

```python
import pandas as pd
import numpy as np
import matplotlib.pyplot as plt
from sklearn.neighbors import NearestNeighbors
from collections import deque

plt.rcParams["font.sans-serif"] = ["SimHei"]
plt.rcParams["axes.unicode_minus"] = False


# =========================
# 1. 读取数据
# =========================

df = pd.read_csv("molecular_interaction_manifest.csv", skiprows=[1])

num_cols = [
    "Manifold_X", "Manifold_Y", "Bioactivity_Score",
    "LogP", "TPSA", "MolWt", "Dipole_Proxy",
    "Max_Partial_Charge", "Balaban_J", "Bertz_CT"
]

for c in num_cols:
    df[c] = pd.to_numeric(df[c], errors="coerce")

df = df.dropna(subset=["Manifold_X", "Manifold_Y", "Bioactivity_Score"]).reset_index(drop=True)

X = df[["Manifold_X", "Manifold_Y"]].values
A = df["Bioactivity_Score"].values
n = len(df)


# =========================
# 2. 参数自适应设置
# =========================
# eps 用第 8 近邻距离的中位数
# MinActivity 用邻域活性总和的 75% 分位数

k = 8
nbrs = NearestNeighbors(n_neighbors=k+1).fit(X)
distances, indices = nbrs.kneighbors(X)

eps = np.median(distances[:, k])

print("eps =", eps)


# =========================
# 3. 构造邻域
# =========================

neighbors = []

for i in range(n):
    d = np.sqrt(np.sum((X - X[i]) ** 2, axis=1))
    neigh = np.where(d <= eps)[0]
    neighbors.append(neigh)

neighbor_activity_sum = np.array([A[neighbors[i]].sum() for i in range(n)])

MinActivity = np.quantile(neighbor_activity_sum, 0.75)

print("MinActivity =", MinActivity)


# =========================
# 4. 活性阈值滤波
# =========================
# 参考文献中的能量阈值滤波，这里使用活性分数的中位数作为初筛阈值

activity_filter = np.quantile(A, 0.50)

high_activity_candidate = A >= activity_filter

print("活性初筛阈值 =", activity_filter)
print("初筛后分子数 =", high_activity_candidate.sum())


# =========================
# 5. E-DBSCAN 聚类
# =========================

labels = np.full(n, -1)       # -1 表示普通点或噪声点
visited = np.zeros(n, dtype=bool)

cluster_id = 0

for i in range(n):

    if visited[i]:
        continue

    visited[i] = True

    # 如果没有通过活性初筛，则不作为热点起点
    if not high_activity_candidate[i]:
        continue

    # 判断是否为核心点
    if neighbor_activity_sum[i] < MinActivity:
        continue

    # 新建一个热点区域
    labels[i] = cluster_id
    queue = deque(neighbors[i])

    while queue:
        j = queue.popleft()

        if not visited[j]:
            visited[j] = True

            # 只让活性较高的点参与热点扩展
            if high_activity_candidate[j] and neighbor_activity_sum[j] >= MinActivity:
                queue.extend(neighbors[j])

        # 如果该点通过初筛，则归入当前热点
        if labels[j] == -1 and high_activity_candidate[j]:
            labels[j] = cluster_id

    cluster_id += 1


df["Hotspot_Label"] = labels
df["Region"] = np.where(df["Hotspot_Label"] >= 0, "Hotspot", "Ordinary")

print(df["Region"].value_counts())
print("热点区域数量 =", cluster_id)


# =========================
# 6. 区域统计比较
# =========================

compare_cols = [
    "Bioactivity_Score", "LogP", "TPSA", "MolWt",
    "Dipole_Proxy", "Max_Partial_Charge",
    "Balaban_J", "Bertz_CT"
]

summary = df.groupby("Region")[compare_cols].agg(["count", "mean", "std", "median"])

print("\n热点区域与普通区域统计比较：")
print(summary)

summary.to_csv("Q1_region_summary.csv", encoding="utf-8-sig")


# =========================
# 7. 各热点区域统计
# =========================

hot_detail = df[df["Region"] == "Hotspot"].groupby("Hotspot_Label")[compare_cols].agg(["count", "mean", "median"])

print("\n各热点区域统计：")
print(hot_detail)

hot_detail.to_csv("Q1_hotspot_detail.csv", encoding="utf-8-sig")


# =========================
# 8. 选代表性分子
# =========================

representatives = []

# 每个热点区域选：活性最高分子 + 区域中心分子
for label in sorted(df[df["Hotspot_Label"] >= 0]["Hotspot_Label"].unique()):

    sub = df[df["Hotspot_Label"] == label].copy()

    # 活性最高
    top = sub.sort_values("Bioactivity_Score", ascending=False).head(1)
    top["Representative_Type"] = "热点内最高活性分子"

    # 离区域中心最近
    cx = sub["Manifold_X"].mean()
    cy = sub["Manifold_Y"].mean()
    sub["dist_center"] = np.sqrt((sub["Manifold_X"] - cx) ** 2 + (sub["Manifold_Y"] - cy) ** 2)
    center = sub.sort_values("dist_center").head(1)
    center["Representative_Type"] = "热点中心代表分子"

    representatives.append(top)
    representatives.append(center)

# 普通区域也选几个代表：活性中等且靠近普通区域中心
ordinary = df[df["Region"] == "Ordinary"].copy()
cx = ordinary["Manifold_X"].mean()
cy = ordinary["Manifold_Y"].mean()
ordinary["dist_center"] = np.sqrt((ordinary["Manifold_X"] - cx) ** 2 + (ordinary["Manifold_Y"] - cy) ** 2)
ordinary_rep = ordinary.sort_values("dist_center").head(3)
ordinary_rep["Representative_Type"] = "普通区域代表分子"
representatives.append(ordinary_rep)

rep = pd.concat(representatives)

rep_cols = [
    "Region", "Hotspot_Label", "Representative_Type",
    "ID", "SMILES", "Manifold_X", "Manifold_Y",
    "Bioactivity_Score", "LogP", "TPSA", "MolWt", "Bertz_CT"
]

rep = rep[rep_cols]

print("\n代表性分子：")
print(rep)

rep.to_csv("Q1_representative_molecules.csv", index=False, encoding="utf-8-sig")


# =========================
# 9. 画图
# =========================

plt.figure(figsize=(8, 6))
plt.scatter(
    df["Manifold_X"], df["Manifold_Y"],
    c=df["Bioactivity_Score"],
    cmap="viridis",
    s=45,
    edgecolor="k",
    linewidth=0.2
)
plt.colorbar(label="Bioactivity Score")
plt.xlabel("Manifold_X")
plt.ylabel("Manifold_Y")
plt.title("二维流形空间中的分子活性分布")
plt.tight_layout()
plt.savefig("Q1_activity_distribution.png", dpi=300)
plt.show()


plt.figure(figsize=(8, 6))

for region, color in zip(["Ordinary", "Hotspot"], ["lightgray", "red"]):
    sub = df[df["Region"] == region]
    plt.scatter(
        sub["Manifold_X"], sub["Manifold_Y"],
        label=region,
        s=55,
        c=color,
        edgecolor="k",
        linewidth=0.2
    )

plt.xlabel("Manifold_X")
plt.ylabel("Manifold_Y")
plt.title("基于活性密度 E-DBSCAN 的热点区域识别")
plt.legend()
plt.tight_layout()
plt.savefig("Q1_hotspot_partition.png", dpi=300)
plt.show()


plt.figure(figsize=(6, 5))
df.boxplot(column="Bioactivity_Score", by="Region")
plt.title("热点区域与普通区域活性得分比较")
plt.suptitle("")
plt.xlabel("区域")
plt.ylabel("Bioactivity Score")
plt.tight_layout()
plt.savefig("Q1_activity_boxplot.png", dpi=300)
plt.show()


# =========================
# 10. 保存结果
# =========================

df.to_csv("Q1_molecules_with_region.csv", index=False, encoding="utf-8-sig")

print("\n问题1完成，结果已保存。")
```

## 1. 模型名称

> 基于活性密度空间聚类的分子热点区域识别模型

或者：

> 改进 E-DBSCAN 的高活性热点区域识别模型

---

## 2. 方法描述

可以写：

参考赵云龙等提出的基于能量密度空间聚类算法的思想，本文将频散谱中的能量值类比为分子的生物活性得分，将频率—速度空间类比为分子的二维流形空间。对于每一个分子点，计算其 \(\varepsilon\) 邻域内的活性总和，若该局部活性密度超过给定阈值，则认为该分子处于高活性密集区域。随后以核心点为起点进行区域扩展，将空间相邻且活性密度较高的分子划分为同一热点区域。与传统 DBSCAN 仅考虑点数量密度不同，该方法同时考虑了分子的空间邻近性和活性大小，因此更适合识别二维流形空间中的高活性热点区域。

---

## 3. 数学定义

设分子集合为：

\[
D=\{p_1,p_2,\cdots,p_n\}
\]

其中：

\[
p_i=(x_i,y_i,A_i)
\]

\[
x_i,y_i
\]

分别表示二维流形坐标，

\[
A_i
\]

表示生物活性得分。

定义 \(\varepsilon\) 邻域：

\[
N_{\varepsilon}(p_i)=\{p_j\in D\mid d(p_i,p_j)\leq \varepsilon\}
\]

其中：

\[
d(p_i,p_j)=\sqrt{(x_i-x_j)^2+(y_i-y_j)^2}
\]

定义邻域活性密度：

\[
S_i=\sum_{p_j\in N_{\varepsilon}(p_i)} A_j
\]

若：

\[
S_i\geq MinActivity
\]

则 \(p_i\) 为核心点。由核心点及其密度可达点构成的连通区域定义为高活性热点区域。其余分子划入普通区域。

# 七、参数选择说明

代码中参数自动选取：

## 1. 邻域半径 \(\varepsilon\)

取每个点第 8 近邻距离的中位数：

\[
\varepsilon=\text{median}(d_{i,8})
\]

这样可以反映数据自身的空间尺度，避免人为指定半径。

## 2. 活性密度阈值 \(MinActivity\)

先计算每个分子的邻域活性总和：

\[
S_i=\sum_{p_j\in N_{\varepsilon}(p_i)} A_j
\]

然后取其 75% 分位数：

\[
MinActivity=Q_{0.75}(S)
\]

这样只有局部活性密度处于较高水平的区域才会被判定为热点区域。

## 3. 活性初筛阈值

参考文献中的“能量阈值滤波”，本文先用活性得分中位数进行初筛：

\[
A_i\geq Q_{0.50}(A)
\]

低于该阈值的分子不参与热点区域扩展。
